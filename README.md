# Resume Analyzer AI

A Streamlit web application that leverages Google Gemini AI to analyze resumes against job descriptions. The app provides ATS scoring, selection probability, fit analysis, and SWOT analysis to help candidates understand their strengths, weaknesses, and areas for improvement.

## Features
- **Resume Upload:** Upload your resume in PDF format.
- **Job Description Input:** Paste the job description for the target role.
- **AI-Powered Analysis:**
  - ATS Score (0–100) with strengths, gaps, matched/missing keywords, and improvement suggestions.
  - Selection Probability (0–100%) with strengths, weaknesses, risk factors, and improvement suggestions.
  - Fit Status (Good Fit / Moderate Fit / Not a Good Fit) with strengths, weaknesses, missing skills/keywords, and suggestions.
  - SWOT Analysis (Strengths, Weaknesses, Opportunities, Threats).

## How It Works
1. Upload your resume (PDF) using the sidebar.
2. Paste the job description in the main text area.
3. Click **Generate Results** to receive a detailed AI-powered analysis.
